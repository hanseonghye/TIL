# vscode editor

### import 단축키

`ctrl + .`

### insert new line

- 아래에 insert

  `ctrl + enter`

- 위에 insert

  `ctrl + shift + enter`

## open in new tab

`settings.json`파일에 다음을 추가

`"workbench.editor.enablePreview": false`



### 복사 붙여넣기

`alt + shift  + 방향키`



### 파일 찾기

`ctrl + p`



### new line 추가

### 위에 줄추가

`shift + ctrl + enter`
