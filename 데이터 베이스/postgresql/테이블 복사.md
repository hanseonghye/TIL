# 테이블 복사

- 테이블 구조만 복사

  ` CREATE TABLE newtable (LIKE oldtable);`

- 테이블 구조와 데이터 복사

  `CREATE TABLE  newtable AS SELECt * FROM oldtable;`

- 테이블, 인덱스, 외래키 등의 정보 다 복사

  ``` sql
  CREATE TABLE newtable (LIKE oldtable INCLUDING ALL);
  INSERT INTO newtable ( select * from oldtable); 
  ```
