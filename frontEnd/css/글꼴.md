# 글꼴

### 사이즈 단위

- px
  - 모든 브라우저의 기본 글꼴의 크기는 16px를 기준으로 한다. 한글 글꼴의 경우 16px이 가장 가독성이 좋기 때문이다.
  - 가장 많이 사용하는 단위
- %
  - 12px = 75% = 0.75em
- em
  - 1em = 16px
- pica
- in
- cm, mm



### px보다 `%`, `em`을 써야하는 이유

px를 사용해서 강제적으로 글꼴의 크기를 정하는 경우, 브라우저에서 글꼴의 크기를 조정하더라고 글꼴이 변하지 않기 때문.

최근 브라우저는 강제적으로 px 도 변환을 해 주지만 `%`나 `em`을 쓰는게 좋다.
