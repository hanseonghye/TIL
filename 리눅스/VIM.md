# VIM :pencil2:

## 입력

### i

커서 앞에 삽입

### o

현재 줄 다음에 삽입

### I

현재 줄 첫 칸 앞에 텍스트 입력

### x

커서 위치의 문자 삭제



## 이동

### :n , nG

n번째 줄로 이동

### :$ , G

파일의 끝 줄로 이동



## 검색

### /문자열

문자열 탐색

### n

다음 문자열 탐색

### N

역방향으로 문자열 탐색



---

### shift+zz

종료

### :!

vi를 잠시 중단하고 지정한 명령 수행 

### :! ls -l

바로 명령어 실행
