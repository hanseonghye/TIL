## 특수문자 치환

1. `[]`로 감싸주는 문자
   - `*` -> `[*]`
   - `+` -> `[+]`
   - `$` -> `[$]`
   - `|` -> `[|]`



2. `\\`를 붙여주는 것들
   - `(` -> `\\(`
   - `)` -> `\\)`
   - `{` -> `\\{`
   - `}` -> `\\}`
   - `^` -> `\\^`
   - `[` -> `\\[`
   - `]` -> `\\]`
