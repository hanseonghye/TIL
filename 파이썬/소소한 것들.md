### print할때 구분 : sep

```python
a = 1
b = a + 1
print(a, b, sep=' ')
```

### print 할때 개행

`end`속성으로 컨트롤 하자.

```python
print("1*1 = 1", end='\t')
```

### 신박한 print

```python
print(f'신박하게 a와 b를 출력하자 [{a}, {b}]')
```



### print err 빨간색으로 출력하기
