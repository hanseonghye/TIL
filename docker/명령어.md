# 명령어

- 이미지 확인
  - ` > docker images`
- 컨테이너 목록 확인
  - `> docker ps 옵션`
- 사용하지 않는 컨테이너 확인
  - `> docker ps -a`
- 사용하지 않는 컨테이너 다시 띄우기
  - `> docker start [컨테이너]`
- 컨테이너에 접근하기
  - `> docker exec -i -t [컨테이너] bash`



### 이미지 만들기

- ` docker build -t [이미지명] .`

`Dockerfile`이 있는 경로에서 해야한다



###  이미지 삭제

`$docker rmi 이미지id`



### 컨테이너 삭제

`$ docker rm 컨테이너id1, id2`



### 컨테이너 실행

`$ docker run -p 8081:8081 -t hanseonghye/springboot-docker`
